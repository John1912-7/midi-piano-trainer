# MIDI Piano Trainer

Личный браузерный тренажер: загружаешь `.mid` файл, видишь падающие ноты и играешь их на клавиатуре ПК.

## Что уже есть

- Загрузка MIDI прямо в браузере, без отправки файла на сервер.
- Парсинг обычных MIDI-файлов формата 0/1 с tempo events.
- Падающие ноты на canvas.
- Простая оценка попаданий: hit/miss/wrong.
- Звук через Web Audio API.
- Выбор дорожки MIDI.
- Сдвиг диапазона клавиатуры по октавам.

## Как запустить локально

Открой `index.html` через локальный веб-сервер или GitHub Pages.

Если установлен Python:

```bash
python -m http.server 8080
```

Потом открой `http://localhost:8080`.

## GitHub Pages

Проект статический, поэтому его можно публиковать через GitHub Pages без backend-сервера.
Файлы MIDI выбираются с твоего компьютера и остаются в браузере.
