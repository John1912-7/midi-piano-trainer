YouTube-ссылка

YouTube-to-MIDI оставляем следующим этапом. Для бесплатного и безопасного MVP сайт не скачивает YouTube-аудио: пользователь загружает свой аудиофайл, а обработка идёт на backend.

Что можно сделать на GitHub Pages

Можно добавить страницу, загрузку аудиофайла, интерфейс настроек и простую обработку прямо в браузере. Это подойдет для экспериментов с короткими файлами, но качество распознавания будет ограниченным.

Что требует backend

Нормальная конвертация песни в MIDI требует тяжелой модели распознавания нот. Для этого нужен сервер: он принимает файл, запускает AI-модель, сохраняет результат и возвращает MIDI в тренажер.

Почему YouTube сложнее

Официальный YouTube API не является инструментом для скачивания аудио из видео. Поэтому надежный и юридически аккуратный вариант - не скачивать YouTube напрямую, а просить пользователя загрузить свой аудиофайл.

Что уже делает MVP

  1. Принимает MP3/WAV/OGG/FLAC с компьютера.
  2. Отправляет файл на backend с AI-моделью Basic Pitch.
  3. Показывает статус: файл принят, идет распознавание, MIDI готов.
  4. Позволяет скачать MIDI или открыть его в текущем тренажере.
  5. YouTube-ссылки оставить как отдельный вопрос после проверки правил и архитектуры.
