How to use the trainer

Everything runs in the browser: your MIDI file does not need to be sent to a server.

  1. 1. Upload MIDI Any .mid or .midi file works. If you only have a piano recording, try Audio-to-MIDI first.
  2. 2. Hear the result Press Listen to hear the MIDI through the built-in piano, or Play to start the falling notes.
  3. 3. Practice the right part Choose a track, adjust the speed, and play using the keys highlighted on the virtual keyboard.
